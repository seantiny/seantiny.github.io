D.U.C.K - A IoT System

Developed a 3-device system using ESP32 microcontrollers and ESP-Now to monitor user presence and phone use in real time. Unlike passive tools, D.U.C.K alerts a third party to encourage focus and habit-building. Designed for low-latency, offline communication without WiFi.

BUCK: Desk unit with a mmWave sensor for presence detection and a servo-based visual timer. Unlike passive monitoring tools, D.U.C.K actively sends alerts via ESP-Now to a third party to promote better study habits. It is housed in a duck enclosure for branding sake.

PUCK: A remote phone-monitoring device that uses a load cell with an HX711 amplifier module to detect phone presence. Sends distraction data to Monitor in real time with custom weight threshold logic.

MONITOR: Handheld device for a third-party accountability partner. Receives alerts via ESP-Now and notifies users with LEDs and a buzzer. Designed for portability and quick response without internet dependency.

Goose Works - Automatic Grabber

GooseGrabber is an electronically powered assistive gripper prototype designed to aid users with limited grip strength or hand aches. It features an extending arm driven by a stepper motor that increases the wingspan by approximately one foot, enabling extended reach.

The gripper fingers are actuated by servo motors, providing precise control for handling various objects. Structural parts are 3D printed in PLA for rapid prototyping, while high-stress components like pins and the gripper base use laser cut plywood for enhanced strength and durability. Flexible TPU parts serve as motor mounts and rod housings, absorbing bending stresses and motor vibrations to improve longevity.

Pomochromo - A Chrome Extension

Collaborated on the full-stack development of a Chrome extension designed to reduce procrastination for high school students using the Pomodoro technique. The extension manages timed work and break intervals to improve focus and productivity.

Led back-end development, integrating Chrome.storage API to implement persistent client-side data storage that efficiently manages user preferences and session states without relying on external backend services. Supported front-end development using JavaScript to build a responsive and intuitive user interface that enables seamless session control and real-time feedback.
